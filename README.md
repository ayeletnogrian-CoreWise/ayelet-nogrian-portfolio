## Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `learning-enablement-portfolio`)
2. Push this entire folder to the repository root
3. Go to **Settings ‚Üí Pages ‚Üí Source ‚Üí Deploy from branch ‚Üí main / root**
4. Update the LinkedIn URL in `index.html` (search for `ADD_LINKEDIN_HERE`)
