## 🌐 Deployment Instructions (GitHub Pages)

1. Create a new GitHub repository, for example: `learning-enablement-portfolio`.
2. Push this entire local directory to the repository root on the `main` branch.
3. In the repository settings, navigate to **Settings → Pages**.
4. Set the source to **Deploy from branch**, select `main` and `/root`, then click **Save**.
5. Ensure the production LinkedIn redirect URL is fully configured inside `index.html`.
